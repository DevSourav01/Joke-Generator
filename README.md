# Joke Generator

A simple joke generator web application built with:

- **React** + **Vite**  
- **TypeScript**  
- **Tailwind CSS**  

Fetches random jokes from a public API and displays them with a clean, responsive UI.

## Stack

- Framework: React + Vite  
- Language: TypeScript  
- Styling: Tailwind CSS  
- API: (e.g., `https://api.chucknorris.io/jokes/random` or any joke API)
